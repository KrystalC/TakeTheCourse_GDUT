# TakeTheCourse_GDUT
# GDUT教务系统选课助手

# 功能

* 支持自动验证码识别登录。
* 支持关键字、优先级、多备用课程的选课方式。
* 支持批量登录、选课、验证操作。
* 支持批量导入用户。

# 使用方法

## **文件 ``Template.xlsx`` 使用方法**

* ``account``  : 教务系统账号
* ``password`` : 教务系统密码
* ``key``      : 所选课程名字（或教师名字）关键字。``,``（英文逗号）: 顺序优先级的与操作 ； ``|`` ：顺序优先级的或操作
  
注意：优先级从左到右
